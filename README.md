# 联想G480四叶草版EFI

## 配置
- CPU `i5-3230M`*（2.6GHZ）*

- 声卡 `ALC-269`

- 显卡 `HD4000`

- 无线网卡 `AR9485`

- 无 蓝牙 *（没有驱动...）*
